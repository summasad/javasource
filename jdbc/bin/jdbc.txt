JDBC(Java Database Connetivity) : 자바와 데이터베이스의 상호 연동하는 방법
-Java 어플리케이션 내에서 JDBC 를 이용해서 데이터베이스에 접근
-정해진 순서

-드라이버 : 데이터베이스에 맞는 드라이버 준비(오라클 - ojdbc버전번호.jar)
-JDBC 프로그래밍 단계
1) 드라이버 로드 : Class.forName(오라클드라이버 이름)
2) 드라이버를 통해 데이터베이스 연결문 작성
    url, user id, user password
3) 1,2 작업이 성공한다면 데이터베이스 서버와 연동하는 커넥션을 하나 얻게 됨
4) 3에서 얻어낸 커넥션을 통해 Data를 가져올 수 있게 됨 
5) ResultSet 객체 사용(Dbeaver 등에서 작업한 결과 행들을 담기 위한 객체) =>SQL문 담아냄 리스트에 담기
