다형성(polymorphism)
- 상속 개념과 연관이 깊다
- 여러가지 형태를 가질 수 있는 능력
- 조상 클래스 타입의 참조변수로 자손 클래스의 인스턴스 참조
- Parent parent2 = new Child();
  - 왼쪽 과 오른쪽이 다를때 다형성
  - 왼쪽은 부모타입 = 오른쪽은 자식타입
  - 상속 관계 일때
  - parent2가 접근할 수 있는 범위가 제한됨. Child()이지만 Parent에 있는 범위만 
  - 자식 = 부모는 오류
    - ClassCastException (실행시(런타임) 오류 발생) : instanceof 연산자로 막아줌
    - if (부모 instanceof 자식)
        명제가 true 라면, 맞는 형식이면

- 캐스팅
  - 자손 타입은 조상 타입으로 자동 형변환 가능
  - 조상 타입은 자손 타입으로 강제 형변환 해야함

- 매개변수의 다형성

- 같은 코드 반복하지 않기 위해 다형성 메소드 활용

- 부모 자식 클래스 설계
- 복수의 클래스에서 공통의 속성을 뽑아 부모 클래스로 설계
- 자식 클래스에서 속성 상속받은 후 개별 특성 정리하면 효율적 

- 상속 시 메소드 오버라이딩 하면
- 부모 메소드는 가려지고 자식 메소드만 실행
- 부모 메소드를 이용해서 오버라이딩한 다양한 자식 메소드를 하나로 처리
