다형성(polymorphism)
- 상속 개념과 연관이 깊다
- 여러가지 형태를 가질 수 있는 능력
- 조상 클래스 타입의 참조변수로 자손 클래스의 인스턴스 참조
- Parent parent2 = new Child();
  - 왼쪽 과 오른쪽이 다를때 다형성
  - 왼쪽은 부모타입 = 오른쪽은 자식타입
  - 상속 관계 일때
  - 자식 = 부모는 오류
    - ClassCastException (실행시(런타임) 오류 발생) : instanceof 연산자로 막아줌
    - if (부모 instanceof 자식)
        명제가 true 라면, 맞는 형식이면

- 캐스팅
  - 자손 타입은 조상 타입으로 자동 형변환 가능
  - 조상 타입은 자손 타입으로 강제 형변환 해야함

- 매개변수의 다형성
