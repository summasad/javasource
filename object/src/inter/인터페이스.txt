인터페이스
- 추상 클래스와 유사
- 추상 클래스보다 추상화 정도가 높다
 **상수만 가능 : 멤버 변수 선언 불가
   - 상수 앞에 public static final 생략가능, 모두 상수라서
 **추상 메소드만 내릴 수 있음 : 일반 메소드 선언 불가
   - 메소드 앞에 abstract 키워드 생략 가능, 모두 추상 메소드라서
   - 일반 메소드 대신 default 나 static 붙이는건 가능

-구현(implements)
   일반 클래스가 인터페이스의 기능을 구현

-상속
   인터페이스가 인터페이스를 상속받음(다중 상속 가능)
    -부모와 자식 모두 인터페이스 여야 함
    *extends와 implements 같이 사용 가능

-익명 구현 객체
  -한번 구현하고 다시 사용하지 않을 거라 하위클래스를 만들지 않은 경우
  remoteControl = new RemoteControl() 퀵 픽스 해서 직접입력
